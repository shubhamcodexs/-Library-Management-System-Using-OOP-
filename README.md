# 📚 Library Management System (Java - OOP, CLI)

## 🎯 Objective
This project is a **Command Line Interface (CLI)-based Library Management System** built using Java. It demonstrates core **Object-Oriented Programming (OOP)** concepts by managing books and users, along with book issue and return functionality.

---

## 🛠️ Tools & Technologies
- Java
- VS Code / IntelliJ IDEA CE
- Terminal / Command Prompt
- Java Collections (ArrayList)

---

## ✨ Features
- ➕ Add new books  
- 👤 Add users  
- 📖 View all books  
- 👥 View all users  
- 🔁 Issue book  
- 🔄 Return book  
- 📋 Menu-driven interface
